== Description ==

HosterGo is a Wordpress plugin providing a full client management and billing solution to hosting companies.

* Billing Ð Multi-currency, value added tax & recurring invoicing
* Payment processing Ð Support for Paypal
* Control panels Ð Integration with cPanel and Kloxo (aka Lxadmin)
* Registrars Ð Integration with Enom
* Automation Ð Account creation, domain registration, order & invoice processing
* Products Ð Offer free & recurring products, custom welcome emails
* Dashboard Ð View key data in one screen

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the HosterGo settings panel in your Wordpress dashboard and install the plugin.
4. Activate the 4 sidebar widgets.
5. Link to the Admininistration Panel of Hoster, login with the default user admin and password admin_1234.
